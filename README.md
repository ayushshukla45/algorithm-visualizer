# Algorithm Visualizer

A simple web-based algorithm visualizer to demonstrate how common sorting algorithms work step by step.

## Features
- Visualizes Bubble Sort and Selection Sort
- Random array generation
- Step-by-step animation for better understanding

## Algorithms Implemented
- Bubble Sort (Time Complexity: O(n²))
- Selection Sort (Time Complexity: O(n²))

## Tech Stack
- HTML
- CSS
- JavaScript

## Learning Outcomes
- Improved understanding of sorting algorithms
- Gained hands-on experience with time complexity and algorithm behavior
- Strengthened problem-solving and debugging skills
